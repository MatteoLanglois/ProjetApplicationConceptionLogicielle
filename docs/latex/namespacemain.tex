\doxysubsection{Description détaillée}
Un programme qui joue au jeu puissance 4++. 

Programme principal du jeu.

Ce programme est un jeu de puissance 4++ avec une grille de taille variable, un nombre de pions à aligner variable, des bonus et un undo.

Ce programme utilise les modules externes suivants \+:
\begin{DoxyItemize}
\item tkinter
\item numpy
\item inspect
\item argparse

Ce module permet de lancer le jeu. Il contient un parser d\textquotesingle{}arguments qui permet de lancer le jeu en ligne de commande ou avec l\textquotesingle{}interface graphique. 
\end{DoxyItemize}
