\doxysubsection{Description détaillée}
Ce module contient les fonctions de base utile à la vue du jeu. 

Ce module permet de facilement avoir le menu sur toutes les fenêtres. De quitter le jeu, de recommencer une partie, etc. Il permet aussi d\textquotesingle{}afficher des messages d\textquotesingle{}information ou d\textquotesingle{}avertissement.
